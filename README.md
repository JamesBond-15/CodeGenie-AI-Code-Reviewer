# CodeGenie – Your AI Code Companion

CodeGenie is a web-based application that helps users understand and improve Python code using AI.  
It is built for both beginners who want clear explanations and developers who want structured code reviews.

Instead of giving generic feedback, CodeGenie adapts how it responds based on the user’s experience level.

---

##  What it does
- Explains Python code in simple language  
- Reviews code and suggests improvements  
- Shows side-by-side comparisons of original and optimized code  
- Allows anonymous usage  
- Provides login-based history and community features  
- Works smoothly on desktop and mobile devices  

---

##  Review Tones
CodeGenie allows users to choose how they want feedback to be given:

| Tone           | Purpose                                                   |
|----------------|-----------------------------------------------------------|
| **Gentle Mentor** | Beginner-friendly explanations and supportive feedback |
| **Zen Master**    | Balanced and thoughtful review style                   |
| **Brutal Senior** | Direct and professional code critique                  |

These tones change the style of AI feedback, making learning more engaging.
